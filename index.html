Project

I have designed, developed and maintained automated test suites using Cypress before but in my opinion, it is always valuable to empty our cup and start from the beginning to demonstrate and strengthen fundamental skills. With that in mind I opted to automate a few scenarios using the website Swag Labs (saucedemo.com). SauceDemo is a sample website built by SauceLabs that allows users to practice browser automation.

The bulk of the project is designed around automating scenarios for SauceDemo and to that end I will focus on discussing those instead. It's also worth noting that this project is very much a work in progress, there is a lot of scope to add new scenarios in the future. Continued iteration is a healthy part of the development process and I expect I will continue to update the code and this post as time goes on.
