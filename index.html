About me

I'm a quality assurance automation engineer from Detroit, Michigan, working in web and application development. I'm an advocate for software quality who loves to help others build and maintain software applications that deliver quality and impactful interactions with their user base.

I have recent experience in e-commerce, SaaS, and sUAV software domains. I'm an energetic, quality-focused, collaborative professional who specializes in creating and executing software test plans, identifying problems with adverse impacts on the value of software projects, and quickly communicating concerns to mitigate risks and minimize negative client impact.

I've built test automation solutions from the ground up. I've also done consulting to help teams with testing and automation. My favorite languages are JavaScript and Python, and my favorite automation framework is Playwright, but I can quickly pick up anything.
